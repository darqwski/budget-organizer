### Budget organizer

#### Frontend - Vite + React

#### Backend - NodeJS + Express + MySQL

### How to run

1. `cd frontend/ && npm run rebuild-tailwind`
2. `cd frontend/ && npm run dev`
3. `npm run dev-server`
